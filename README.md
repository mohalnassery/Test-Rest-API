# Inventory Reservation API

built with raw node + sqlite.

## Run

1. npm install
2. npm start

## Test

npm run test:scenario

## Endpoints

```sh
POST /resources (body: name, total_capacity)
GET /resources/:id
POST /resources/:id/reservations (body: reserver_name, quantity)
GET /resources/:id/reservations
```
